Hello World

Das erste Programm, welches man in einer Programmiersprache schreibt, ist typischerweise das Hello World.

Bevor wir den Code schreiben, müssen wir jedoch noch kurz auf die Kommandozeile, um in einem Ordner unserer Wahl ein Go-Modul zu initiieren. Das kann beispielsweise im Terminal von VSCode geschehen:

go mod init helloworld

Dieser Befehl generiert eine go.mod-Datei im aktuellen Verzeichnis. In dieser wird alles gespeichert, was für unser Projekt relevant ist (Go-Versionsnummer, externe Abhängigkeiten, ...). Aber dazu später mehr. Für Go ist nur wichtig, dass diese Datei existiert.

Die ersten paar Zeilen Go

Erzeugen wir nun eine Datei mit dem Namen main.go und füllen sie mit folgendem Inhalt: 

package main

import "fmt"

func main() {
    fmt.Println("Hallo Welt!")
}

Was die einzelnen Zeilen Code bedeuten, gehen wir gleich genauer drauf ein. Zunächst können wir unser Hello-World-Programm mit dem folgenden Befehl ausführen:

go run .

In der Kommandozeile erhalten wir unser freundliches "Hallo Welt!". Fantastisch!

Was bedeutet das alles?

Gehen wir aber nun auf die verschiedenen Teile des Programms ein.

package main

Ganz oben in jeder Quellcode-Datei wird das aktuelle Paket (package) benannt. Go-Pakete - auch Module genannt - basieren auf Ordnern und werden im besten Fall analog zum Ordnernamen benannt. Alle Dateien im gleichen Ordner haben die gleiche erste Zeile. 

import "fmt"

Nach dem Paketnamen folgen die ganzen Importe. Diese können aus eigenen Paketen, aus externen Libraries oder, wie in unserem Code, aus der Go-Standardbibliothek sein.

Das Go-Paket fmt beinhaltet verschiedene Funktionen, um Text zu formatieren und ausgeben zu können. Weil wir in unserem Programm etwas ausgeben wollen, müssen wir das Paket importieren. 

func main() {
    fmt.Println("Hallo Welt!")
}

In Zeile 5 definieren wir mit dem Keyword func eine Funktion mit dem Namen main. Sie besitzt keine Übergabeparameter. Das liegt unter anderem daran, dass die main-Funktion in Go eine besondere Bedeutung hat. Führt man eine Go-Datei aus, ist in dieser Datei die main-Funktion der Einstiegspunkt.

In Zeile 6 nutzen wir die Funktion Println aus dem Paket fmt, um unser "Hallo Welt!" auf die Kommandozeile zu schreiben. Das war's auch schon. Ganz anders als in Java, aber ähnlich wie in Python wird nicht viel Klimm-Bimm benötigt, um ein lauffähiges Programm zu erhalten.
