Funktionen

Funktionen sind in Go - wie vieles anderes - ebenfalls nicht sehr besonders. Schauen wir uns ein paar Beispiele an. 

func addNumbers(firstNumber int, secondNumber int) int {
  return firstNumber + secondNumber
}

Bei den Übergabeparametern folgt nach dem Variablennamen nocht der Datentyp. Darauf folgt der Typ des Rückgabewerts.

Es ist möglich, mehrere Rückgabewerte zu haben. Den typischen Use-Case dafür lernen wir in einem der folgenden Kapitel kennen. Hier zunächst ein an den Haaren herbeigezogenes Beispiel: 

func createCombinations(w1 string, w2 string) (string, string) {
  return fmt.Sprintf("%s%s", w1, w2), fmt.Sprintf("%s%s", w2, w1)
}

func main() {
  a, b := createCombinations("rilla", "go")
  fmt.Println(a, b)
  // Output:
  // rillago gorilla
}
