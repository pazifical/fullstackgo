Was ist Full Stack Go?

Der Name ist Programm. Hier geht es um Full Stack Web Development mit der Programmiersprache Go - mindestens im Backend.

Im Rahmen meiner Einführungs-Seminar in die Programmiersprache Go ist mir immer wieder aufgefallen, dass es erstaunlich wenig deutschsprachiges Lernmaterial für Go gibt. Das soll diese Webseite lösen. Geplant ist ein Buch, welches Schritt für Schritt eine Full Stack Web-Anwendung in Go lehrt. Zusätzlich werden weitere Themen in einem Blog vertieft.
